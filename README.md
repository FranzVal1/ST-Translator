# SillyTavern Safe Translation

Расширение для SillyTavern 1.18.0. Переводит только текстовые сегменты сообщения через штатные маршруты Google, Yandex или Bing, не отправляя переводчику HTML-теги, макросы, код, URL и встроенные функции.

## Установка

1. Распакуйте папку в `data/<пользователь>/extensions/SillyTavern-Safe-Translation` либо установите ZIP через интерфейс установки сторонних расширений, если ваша сборка поддерживает импорт архива.
2. Перезапустите SillyTavern.
3. Откройте Extensions → Safe Translation.
4. Отключите автоматический режим встроенного Chat Translation, чтобы два переводчика не работали одновременно.

## Поддержано

- Google `/api/translate/google`
- Bing `/api/translate/bing`
- Yandex `/api/translate/yandex`
- Автоперевод ответов персонажа
- Ручная кнопка на сообщениях
- Перевод текущего чата
- Сохранение оригинала в `message.mes`
- Перевод в `message.extra.display_text`
- Защита HTML, макросов `{{...}}`, code blocks, inline code, URL, script/style/pre/code/svg/math

## Ограничение 1.0.0

Текстовые сегменты переводятся последовательно отдельными запросами. Это сделано намеренно: так Google/Yandex/Bing не могут переставить разделители и повредить структуру сообщения.


## 1.0.1

Исправлено ложное отклонение перевода, когда переведённый текст содержал символы `<`, `>` или конструкции, похожие на макросы.
